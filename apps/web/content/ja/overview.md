# サービス概要

securePDF は、PDF の並べ替え・削除・回転・反転・結合と、画像や Office ファイルの PDF 化を扱う編集ツールです。

PDF、JPEG、PNG の通常操作はブラウザ内で完結します。docx、xlsx、pptx のようにバックエンド変換が必要なファイルだけ、ユーザー操作の時点で認証し、変換サーバーへ送信します。

## 主な機能

- PDF ページの選択、並べ替え、削除、回転、反転、挿入。
- JPEG / PNG 画像を PDF ページとして追加し、既存 PDF とまとめて出力。
- docx / xlsx / pptx を PDF に変換して取り込み。
- 右側プレビューでページをスクロール移動し、左側サムネイルと現在ページを同期。
- Web UI、CLI、HTTP API で同じ操作スキーマを共有。

## 処理方式

PDF / JPEG / PNG の編集と出力はブラウザ上で実行します。ファイルを選んだだけでサーバーへアップロードされる設計ではありません。

Office ファイルはブラウザだけでは安定変換できないため、変換が必要になった時点で認証を行い、Cloudflare Worker 経由で変換バックエンドへ送信します。

## 画面構成

- 左側: ページ一覧、ページ選択、ドラッグ移動、挿入位置の指定。
- 右側: PDF ビューワーに近いプレビュー、スクロールによるページ移動。
- 上部: ファイル名、ページ番号、ズーム、印刷、ダウンロード、認証、ドキュメント。

## 対応ファイル

| 種類 | 入力 | 処理場所 |
| --- | --- | --- |
| PDF | `.pdf` | ブラウザ |
| 画像 | `.jpg`, `.jpeg`, `.png` | ブラウザ |
| Office | `.docx`, `.xlsx`, `.pptx` | 認証後の変換バックエンド |

## 提供形態

Web UI は Cloudflare Worker Static Assets で配信します。軽量 API と OpenAPI 仕様は Worker が返し、重い変換処理は Cloud Run などの専用バックエンドへ分離します。
