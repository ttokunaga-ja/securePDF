# サービス概要

securePDF は、ブラウザで PDF を整理して、そのまま新しい PDF として保存できるツールです。

PDF のページ順を変える、不要なページを消す、向きを直す、画像を追加する、といった通常の編集はブラウザ内で完結します。Word、Excel、PowerPoint ファイルを PDF に変換して取り込みたい場合だけ、認証したうえで変換サーバーを使います。

## できること

- PDF ページの並べ替え、削除、回転、反転。
- PDF の途中や末尾へのページ追加。
- JPEG / PNG 画像を PDF ページとして追加。
- docx / xlsx / pptx を PDF に変換して取り込み。
- 右側のプレビューをスクロールしながら、左側のページ一覧で現在位置を確認。
- 編集後の PDF を印刷またはダウンロード。

## 基本的な使い方

1. 画面中央をクリックするか、PDF / 画像 / Office ファイルをドラッグして開きます。
2. 左側のページ一覧で、ページを選択したり並べ替えたりします。
3. 右側のプレビューで、ページの見え方と順番を確認します。
4. 必要に応じてファイル名を変更し、印刷または PDF ダウンロードを実行します。

## ファイルはアップロードされますか

PDF、JPEG、PNG の編集では、ファイルはブラウザ内で処理されます。ページ編集やダウンロードのために、PDF 本文が securePDF のサーバーへ送信されることはありません。

Office ファイルの変換だけは例外です。docx / xlsx / pptx はブラウザ内だけでは安定して PDF 化できないため、変換が必要になった時点で Google 認証または API キーを使い、ファイルを変換サーバーへ送信します。

## 認証が必要な場面

PDF や画像だけを編集する場合、認証は不要です。

Office ファイルを PDF に変換する場合は、API キーが必要です。API キーは、右上の「その他」メニューから「認証」を開いて入力できます。未設定のまま Office ファイルを選ぶと、Google 認証の別ウィンドウで API キーを取得する流れになります。

Office 変換は 1 回あたり 5 クレジットを消費します。クレジットが不足している場合は、変換は実行されません。

## 対応ファイル

| 種類 | 拡張子 | 認証 |
| --- | --- | --- |
| PDF | `.pdf` | 不要 |
| 画像 | `.jpg`, `.jpeg`, `.png` | 不要 |
| Office | `.docx`, `.xlsx`, `.pptx` | 必要 |

## うまくいかないとき

- Office 変換で認証エラーが出る場合は、右上の「その他」メニューから「認証」を開き、API キーを確認してください。
- クレジット不足の表示が出る場合は、その日の変換上限に達しています。
- PDF や画像が開けない場合は、ファイル形式が対応範囲内か、破損していないかを確認してください。
